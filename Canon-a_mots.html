Instructions

Le but du jeu est de taper les mots qui flottent à l'écran pour les détruire avec le canon.

Sur ordinateur utilisez les flèches directionnelles de votre clavier pour vous déplacer.

Sur Mobile utilisez les boutons gauche et droite pour vous déplacer.

Attention: Les mots doivent être obligatoirement écrit en MAJUSCULE.

Have fun !
